## Labels of the activities

* `WALKING` - value = 1: the person was walking
* `WALKING_UPSTAIRS` - value = 2: the person was walking up a staircase
* `WALKING_DOWNSTAIRS` - value = 3: the person was walking down a staircase
* `SITTING` - value = 4: the person was sitting during the test
* `STANDING` - value = 5: the person was standing during the test
* `LAYING` - value = 6: the person was laying down during the test
